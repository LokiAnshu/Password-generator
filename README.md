# PASSWORD GENERATOR
This is a python program which generates a random password according to the user preferences
# Features
Allows the user to select:
  - the length of the password
  - to include only the upper case or lower case or both types of alphabets
  - whether to include special characters or not
  - whether to include numbers or not
# Learning
  - dealing with multiple inputs and variables
  - randomizing the password
  - use of different random methods
  - make the code readable
  - use of lists rather than strings to make the code systematic
